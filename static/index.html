💡 How AI Predictions Work

Plain Language Summary

The AI watches each power line's load, weather, and history to guess how busy it will get. When a line looks overloaded, the system reroutes power before a blackout happens.

Technical Details

Random Forest Regressor trained on 5,000 synthetic samples. Features: historical load, weather, voltage, current, hour, renewable generation, previous congestion.

🛡️ Self-Healing: Lines above 85% congestion are automatically isolated and power is rerouted via A*.

⚡ Control Center Tour

🌐

Welcome to the Control Center

This is a real-time AI-powered grid operations dashboard. It predicts overloads and automatically reroutes power to prevent blackouts.

🎨

Node & Line Colors

  • Green circles = Power plants
  • Blue diamonds = Substations
  • Orange circles = Consumers
  • Lines: 🟢 Healthy → 🟡 Moderate → 🔴 Congested → ⚫ Failed
  • 🔵 Bright blue pulsing = Active power route
▶

Start a Simulation

Use the bottom control panel to Start the simulation. Try preset scenarios like Heatwave or Storm. Watch lines change color as load shifts.

🛡️

Self-Healing & Alerts

When a line exceeds 85% capacity, it trips automatically and power reroutes. Alerts appear in the Event Log and Timeline. Double-click any line to manually test failures.

🤖

Beginner Mode & AI Assistant

Toggle Beginner mode for plain-language labels. Click the 🤖 button to ask the AI assistant questions about what's happening on the grid.
